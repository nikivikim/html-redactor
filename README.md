# html-redactor

**Описание**
---
Это приложение, в котором можно, редактировать HTML страницу, редактировать txt файл. так же реализован сценарий:
Пользователь в поле ввода указывает URL с которого
загружается стартовая страница в средство предпросмотра и html-редактор.
Пользователь может сохранить страницу на своем компьютере как есть или
измененную.

**Функционал приложения:**

- Загрузить URL
- Открыть фйл
- Сохранить файл
- редактировать код
- Применить изменения в коде
- Применить изменения в редакторе
- Редактировать файл


**Технология**
---
Язык программирования Java.
